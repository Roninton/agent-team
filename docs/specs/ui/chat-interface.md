# 群聊界面规范

## 1. 目的与范围
本文档定义VS Code多AI协作平台的群聊界面设计规范，包括布局结构、消息样式、交互逻辑、功能模块等。群聊界面是用户与多个AI代理协作的核心入口，必须遵循本规范实现。

## 2. 布局结构
群聊界面采用三栏式布局，从左到右依次为：
```
┌──────────────────────────────────────────────────────────────────┐
│  [1] 会话列表栏  │  [2] 消息内容区  │  [3] 上下文/任务信息栏      │
│  (240px)         │  (自适应)        │  (320px)                    │
└──────────────────────────────────────────────────────────────────┘
```

### 2.1 左侧会话列表栏（宽度240px，可折叠）
**结构从上到下：**
1. 顶部操作区（高度48px）
   - 左上角：扩展图标 + "AI协作平台"标题
   - 右上角："新建会话"按钮（+图标）、"创建群组"按钮（用户组图标）
2. 搜索框（高度36px，上下间距8px）
   - 搜索图标 + 占位文本："搜索会话..."
   - 右侧清除按钮（输入内容时显示）
3. 会话列表（剩余高度，滚动）
   - 分组："最近会话"、"我的群组"、"单代理会话"
   - 每个会话项高度：48px
   - 会话项内容：头像（32px）、标题、最后消息预览、时间戳、未读消息角标

### 2.2 中间消息内容区（宽度自适应，弹性布局）
**结构从上到下：**
1. 顶部导航栏（高度48px）
   - 左侧：会话/群组头像、名称、成员列表预览（小头像排列）、在线状态
   - 右侧："成员管理"按钮、"会话设置"按钮、"折叠侧边栏"按钮
2. 消息列表区（剩余高度，滚动）
   - 消息气泡布局，支持向上加载历史消息
   - 底部显示"正在输入..."状态提示
3. 底部输入区（高度最小80px，最大300px，随内容自适应）
   - 工具栏：附件按钮、表情按钮、代码块按钮、@成员按钮、快捷指令按钮
   - 输入框：支持多行输入，Markdown语法提示
   - 发送按钮："发送"（Enter/Ctrl+Enter发送提示）、"停止"按钮（生成中显示）

### 2.3 右侧信息栏（宽度320px，可折叠）
**结构从上到下：**
1. 标签页切换（高度36px）
   - 标签："上下文"、"任务列表"、"成员信息"、"设置"
2. 内容区（剩余高度，滚动）
   - 对应标签页的内容，根据选中标签动态切换

## 3. 消息样式规范
### 3.1 消息气泡通用规则
- 最大宽度：消息内容区宽度的75%
- 内边距：12px 16px
- 圆角：8px
- 字体大小：14px，行高1.5
- 间距：消息之间上下间距16px

### 3.2 不同发送者的消息样式
#### 用户消息
- 对齐方式：右对齐
- 气泡背景：主色（#007ACC）
- 文字颜色：白色
- 头像位置：气泡右侧
- 时间戳位置：气泡左下角，灰色小字

#### 代理消息
- 对齐方式：左对齐
- 气泡背景：主题背景色（浅色#F3F3F3 / 深色#252526）
- 文字颜色：一级文本色
- 头像位置：气泡左侧
- 发送者名称：气泡上方，12px灰色文字，显示代理名称
- 时间戳位置：气泡右下角，灰色小字

#### 系统消息
- 对齐方式：居中对齐
- 背景：透明
- 文字颜色：三级文本色
- 字体大小：12px
- 样式：灰色圆角背景（#EAEAEA / #3C3C3C），内边距4px 12px
- 示例："你加入了群组"、"代理Claude Code已上线"、"任务已分配给GitHub Copilot"

### 3.3 消息内容类型样式
#### 文本消息
- 支持Markdown渲染：标题、加粗、斜体、列表、链接、引用等
- 链接颜色：主色，hover下划线
- 代码片段：行内代码使用等宽字体，灰色背景，圆角2px，内边距2px 4px

#### 代码块消息
- 语言标识：右上角显示语言名称（如"TypeScript"）
- 代码内容：使用等宽字体，语法高亮，与VS Code编辑器主题一致
- 工具栏：右上角显示"复制"、"保存到文件"、"在编辑器中打开"按钮
- 行号：左侧显示行号
- 内边距：16px，背景色与编辑器背景一致

#### 工具调用消息
- 样式：灰色边框，左蓝色边框（3px）
- 头部：显示工具名称、状态（运行中/成功/失败）、耗时
- 参数：显示调用的参数，格式化展示
- 结果：折叠/展开展示返回结果，长结果默认折叠
- 状态标识：运行中显示加载动画，成功显示绿色对勾，失败显示红色叉号

#### 附件消息
- 文件卡片：圆角8px，边框，内边距12px
- 内容：文件图标、文件名、大小、操作按钮（下载/打开/保存）
- 支持的附件类型：代码文件、图片、文档、压缩包等
- 图片附件：直接预览显示，支持点击放大查看

#### 任务消息
- 样式：任务卡片，边框，内边距16px
- 内容：任务标题、状态、优先级、执行者、进度条、操作按钮
- 状态更新：任务状态变化时自动更新卡片内容
- 点击：点击卡片跳转到右侧任务详情页

####  markdown表格
- 边框：1px边框，与文本颜色一致
- 表头：背景色稍深，字体加粗
- 单元格：内边距8px，对齐方式根据内容自动调整

## 4. 交互逻辑规范
### 4.1 消息交互
- 消息hover：显示操作工具栏（复制/回复/引用/编辑/撤回/删除）
- 引用回复：点击回复按钮，输入框上方显示引用的消息内容
- 编辑消息：用户可以编辑自己发送的消息，编辑后显示"已编辑"标识
- 撤回消息：消息发送后2分钟内可以撤回，撤回后显示"你撤回了一条消息"
- 消息复制：点击复制按钮，自动复制消息的纯文本内容
- 代码块复制：点击代码块的复制按钮，复制完整代码内容
- 长消息折叠：超过300px高度的消息自动折叠，显示"展开全文"按钮

### 4.2 输入框交互
- 输入@：弹出成员选择列表，支持搜索过滤，选择后@对应成员
- 输入/：弹出快捷指令列表，支持搜索过滤，选择后自动填充指令
- 粘贴内容：粘贴图片直接作为附件发送，粘贴文本保留格式
- 拖放文件：拖放文件到输入区，自动添加为附件
- 快捷键：
  - Enter：发送消息（可设置为Ctrl+Enter发送，Enter换行）
  - Ctrl+Enter：换行（根据设置切换）
  - Shift+Enter：换行
  - Ctrl+Up/Down：切换历史输入记录
  - Tab：补全指令/成员名称

### 4.3 会话列表交互
- 点击会话项：切换到对应会话
- 会话项hover：显示"置顶"、"删除"按钮
- 右键菜单：显示操作菜单（置顶/取消置顶、标记已读/未读、删除会话、重命名）
- 未读消息：显示蓝色小圆点或数字角标，加粗标题
- 会话排序：按最后消息时间倒序排列，置顶会话排在最前面

### 4.4 成员交互
- 点击成员头像：查看成员详情、发送私信、@成员
- 在线状态：头像右下角显示状态标识（绿色=在线，灰色=离线，黄色=忙碌）
- 输入状态：成员正在输入时，在消息列表底部显示"XXX正在输入..."

## 5. 功能模块规范
### 5.1 上下文管理模块（右侧栏）
- 上下文列表：显示当前会话关联的所有文件、代码片段、历史消息
- 文件卡片：显示文件名、路径、修改时间，支持打开、移除、更新
- 添加上下文：支持拖拽文件、手动选择文件、粘贴代码片段
- 上下文同步状态：显示同步状态（已同步/同步中/同步失败）
- 版本信息：显示当前上下文版本号，支持查看历史版本

### 5.2 任务管理模块（右侧栏）
- 任务列表：显示当前群组的所有任务，按状态分组（待分配/执行中/已完成/已失败）
- 任务卡片：显示标题、优先级、执行者、进度、剩余时间
- 任务操作：支持点击查看详情、取消任务、重试任务、分配任务
- 任务筛选：支持按状态、优先级、执行者筛选任务
- 统计信息：顶部显示任务总数、完成率、平均执行时间

### 5.3 成员管理模块（右侧栏）
- 成员列表：显示群组所有成员，按角色排序（所有者/管理员/普通成员）
- 成员卡片：显示头像、名称、角色、在线状态、当前任务数
- 成员操作：支持@成员、查看详情、移除成员、修改角色（管理员权限）
- 邀请成员：支持添加代理到群组、邀请用户加入
- 权限设置：管理员可以设置群组权限（谁可以发送消息、谁可以创建任务等）

### 5.4 会话设置模块（右侧栏）
- 基础设置：会话名称、头像、描述
- 通知设置：消息通知开关、@所有人通知、提及通知、任务更新通知
- 代理配置：默认代理、任务分配策略、上下文同步规则
- 权限设置：成员权限、是否允许邀请新成员、是否允许修改群信息
- 会话操作：清空聊天记录、导出聊天记录、解散群组（所有者权限）

## 6. 状态提示规范
### 6.1 加载状态
- 加载历史消息：列表顶部显示加载动画和"加载中..."提示
- 发送消息中：消息气泡显示加载动画，状态为"发送中"
- 代理生成回复：消息气泡显示打字动画，状态为"正在生成..."
- 工具调用中：工具调用卡片显示加载动画，状态为"运行中..."

### 6.2 状态提示
- 网络断开：顶部显示红色提示条"网络连接断开，正在重连..."
- 重连成功：显示绿色提示"网络已恢复"
- 消息发送失败：消息气泡显示红色感叹号，hover显示失败原因，支持重发
- 任务完成：右下角弹出成功提示，包含任务结果摘要
- 新消息通知：非当前会话有新消息时，显示系统通知，点击跳转到对应会话

### 6.3 空状态
- 空会话：显示欢迎信息，包含快速开始指引、常用功能入口
- 无消息：显示"还没有消息，发送第一条消息开始对话吧"
- 无任务：显示"还没有任务，创建第一个任务开始协作吧"
- 搜索无结果：显示"没有找到相关结果，换个关键词试试"

## 7. 响应式适配
### 7.1 窄屏适配（窗口宽度<1000px）
- 自动折叠右侧信息栏
- 消息气泡最大宽度调整为85%
- 会话列表宽度缩小为200px

### 7.2 超窄屏适配（窗口宽度<600px）
- 自动折叠左侧会话列表和右侧信息栏，只保留消息内容区
- 提供侧边栏展开按钮，点击后侧边栏悬浮显示
- 输入框工具栏简化，只保留常用按钮

### 7.3 大屏适配（窗口宽度>1600px）
- 右侧信息栏宽度扩展到360px
- 消息气泡最大宽度调整为70%
- 支持分栏显示更多内容

## 8. 快捷键支持
| 快捷键 | 功能 |
|--------|------|
| Ctrl + N | 新建会话 |
| Ctrl + Shift + F | 搜索会话 |
| Ctrl + Tab | 切换到下一个会话 |
| Ctrl + Shift + Tab | 切换到上一个会话 |
| Ctrl + F | 在当前聊天中搜索 |
| Ctrl + S | 保存当前上下文 |
| Ctrl + Shift + C | 复制最新回复 |
| Esc | 取消当前输入 / 关闭弹窗 / 停止生成 |
| Ctrl + Enter | 发送消息（可配置） |
| Enter | 发送消息（可配置） |
| Shift + Enter | 换行 |
| @ | 提及成员 |
| / | 打开快捷指令列表 |

## 9. 无障碍支持
- 消息支持屏幕阅读器朗读，包含发送者、内容、时间
- 所有交互元素有明确的aria-label
- 支持键盘导航：Tab键切换消息和按钮，Enter触发操作
- 高对比度模式支持：所有元素在高对比度主题下清晰可辨
