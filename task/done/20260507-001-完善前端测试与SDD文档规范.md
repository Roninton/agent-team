## 技术规范更新

### id命名规范
- 使用标准HTML `id` 属性而非 `data-testid`
- 命名格式：`[页面名]-[元素类型]-[功能描述]`
- 示例：
  - `settings-add-agent-btn` - 设置页添加代理按钮
  - `group-item-1` - 群组列表第一项
  - `settings-agent-modal` - 代理配置模态框

### 测试定位优先级
1. `getElementById` - 最稳定，不随文本/样式变化
2. `getByRole` - 语义化，适合标准控件
3. `getByLabelText` - 表单元素
4. `getByText` - 文本内容
5. `querySelector` - 最后手段

### npm脚本更新
```json
"test": "npm run test:web",
"test:web": "cd web &amp;&amp; npm run test",
"test:src": "vitest run --exclude \"web/**\"",
```
