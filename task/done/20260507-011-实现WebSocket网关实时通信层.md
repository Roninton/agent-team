   - `@WebSocketGateway()` 装饰器配置

1. **系统事件**

   - `chat.message.send` - 接收客户端消息

   - `agent.status.changed` - 代理状态变更推送

   - `group.member.joined` - 成员加入通知
   - `group.member.left` - 成员离开通知

## 核心类与接口

### WebSocketGateway
```typescript
@WebSocketGateway({
  cors: { origin: '*' },
  path: '/ws/v1',
  pingInterval: 30000,
  pingTimeout: 90000,
})
export class WsGateway implements OnGatewayInit, OnGatewayConnection, OnGatewayDisconnect {
  // 客户端连接管理
  private clients: Map<string, Socket> = new Map();
  
  // 消息处理方法
  @SubscribeMessage('chat.message.send')
  handleMessage(client: Socket, payload: any): WsResponse;
  
  @SubscribeMessage('ping')
  handlePing(client: Socket): WsResponse;
}
```
